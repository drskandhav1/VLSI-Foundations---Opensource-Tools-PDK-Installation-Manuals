# README – Open-Source VLSI Tool Setup Flow

## Document Purpose
Master guidance note for students

## Intended Use
Read this document before using any of the detailed manuals.

---

## 1. Purpose

This document is the starting point for students using the open-source VLSI tool setup flow. It explains the order in which the manuals must be followed.

The complete flow has been split into separate documents to keep the setup clear, manageable, and easy to troubleshoot.

---

## 2. Document Usage Order

Students must use the documents in the following order:

### Step 1: Tool Installation Manual
**`OS Tools_IntsallationManual.pdf`**

This document covers installation of the required open-source tools such as:
- xschem
- ngspice
- Magic
- related utilities

### Step 2: PDK Installation and Tool Integration Manual
**`PDK Installation and Tool Integration Manual.pdf`**

Use this document only after completing the tool installation successfully.

This document covers:
- PDK setup
- environment variable configuration
- integration of the tools with the PDK

---

## 3. Important Instructions

- Do not skip stages.
- Do not start PDK integration before completing tool installation.
- Each stage depends on the successful completion of the previous stage.

---

## 4. Troubleshooting Guidance

If any issue occurs, first check the following:

- Whether the correct manual is being used
- Whether the previous stage was completed successfully
- Whether the troubleshooting section in that manual has been reviewed

---

## 5. Recommended Approach for Students

Before starting any setup activity:

1. Read this README fully.
2. Complete the tool installation manual first.
3. Verify that all tools are installed and working.
4. Then proceed to the PDK installation and tool integration manual.
5. Refer to the troubleshooting section whenever an issue occurs.

---

## 6. Note

This README is only a guidance document for flow order and usage.  
For detailed installation steps, configuration commands, and troubleshooting, always refer to the respective manuals.
